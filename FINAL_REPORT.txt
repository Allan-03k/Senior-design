================================================================================
🔧 技术实现
================================================================================

后端技术栈:
- Flask 3.0.0 (Web 框架)
- Anthropic Claude 3.5 Sonnet (AI 模型)
- PIL/Pillow 11.0.0 (图像处理)
- FFmpeg (视频编码)
- SQLAlchemy (ORM)
- Docker (容器化)

前端技术栈:
- React 18+ (UI 框架)
- CSS3 (响应式设计)
- Fetch API (HTTP 客户端)

核心流程:
1. 用户输入食谱信息 (名称、食材、步骤)
2. API 收到请求
3. Claude AI 生成详细步骤 (JSON)
4. PIL 为每个步骤生成图片
5. FFmpeg 合成视频文件
6. 返回视频 URL 到前端
7. 前端播放视频和显示步骤

🐛 已知问题 & 解决方案
================================================================================

问题1: FFmpeg 未安装
解决: Docker 中已预装，或在本地运行:
   Mac: brew install ffmpeg
   Linux: sudo apt-get install ffmpeg

问题4: 端口被占用
